# Hey, I'm VonBraunZ

I'm a self-taught developer working across game modding, legacy Java and industrial tooling. I port, extend, and build things that fill gaps, usually because something didn't exist or didn't work. I've ported Single Player Tarkov mods from TypeScript to C#, backported modern Minecraft mod concepts down to 1.7.10 for GT: New Horizons, built real-time fleet telemetry dashboards for industrial field operations, built a oilfield wellbore visualizer and simulator based on well conditions and built a data visualization tool for oilfield job data that was acquired and taken internal by a corporate buyer.

## How I Work

I pick up mods and tools that are broken or incomplete and make them usable.  
Iterative, run it, read the error, fix it, repeat.  
No unnecessary abstractions. No build steps if I can avoid them.
